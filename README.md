# Análise de Vendas - Dashboard Power BI

Dashboard interativo desenvolvido no Power BI para análise de performance comercial entre 2022 e 2023.

## 🎯 Objetivo do Projeto
Criar um dashboard completo e intuitivo para acompanhar os principais KPIs de vendas, permitindo uma visão clara da receita, ticket médio, volume de pedidos e comportamento de pagamento dos clientes.

## 📸 Visual do Dashboard

![Análise de Vendas Dashboard](Imagem/dashboard-principal.png)

## 📊 Principais Visualizações
- **KPIs Principais**: Receita Total (R$ 2 Bi) e Ticket Médio (R$ 48.820)
- Quantidade de Pedidos Mensal
- Receita por Categoria de Produto
- Distribuição de Métodos de Pagamento

## 🛠️ Tecnologias Utilizadas
- Power BI Desktop
- DAX (medidas calculadas)
- Modelagem de dados
- Visualizações interativas

## 🚀 Como Visualizar
1. Acesse a pasta [`Dashboard/`](Dashboard/)
2. Baixe o arquivo `.pbix`
3. Abra com o **Power BI Desktop**
4. Interaja com os filtros de Ano, Mês e Categoria
